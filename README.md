# Fault-Distance: 1D-CNN для определения расстояния до КЗ

Проект предсказывает расстояние до места короткого замыкания (КЗ) по осциллограммам токов и напряжений с использованием 1D-CNN / ResNet1D.

---

```
Fault-Distance/

└── README.md
```

## Формат входных данных

Каждый файл CSV — **один осциллограф** (одно событие КЗ).

### Структура CSV-файла

| Столбец | Описание | Единица |
|---|---|---|
| `distance_km` | Расстояние до КЗ (одинаково во всех строках файла) | км |
| `CT1IA` | Мгновенные значения тока фазы A | А |
| `CT1IB` | Мгновенные значения тока фазы B | А |
| `CT1IC` | Мгновенные значения тока фазы C | А |
| `S1) BUS1UA` | Мгновенные значения напряжения фазы A | кВ |
| `S1) BUS1UB` | Мгновенные значения напряжения фазы B | кВ |
| `S1) BUS1UC` | Мгновенные значения напряжения фазы C | кВ |

Количество строк = количество временных отсчётов (рекомендуется `SEQ_LENGTH = 400`).

## Быстрый старт

```bash
# 1. Установить зависимости

# 2. Положить CSV-файлы в
data/data_training/

# 3. Запустить обучение
python train.py

# Или с параметрами
python train.py --model cnn1d --epochs 100 --batch-size 32

```

## Архитектуры моделей

| Модель | Входной тензор | Описание |
|---|---|---|
| `cnn1d` | `(B, 6, 400)` | 3-блочный 1D-CNN, быстро обучается |
| `dilated_cnn1d` | `(B, 6, 400)` | Расширенные свёртки, большой receptive field |
| `resnet1d` | `(B, 6, 400)` | ResNet с SE-блоками, лучшая точность |

## Ключевые параметры (`config.py`)

```python
NUM_CHANNELS = 6       # Ia, Ib, Ic, Ua, Ub, Uc
SEQ_LENGTH   = 400     # Временных отсчётов на файл
DATA_DIR     = 'data/data_training'  # Папка с CSV (изолирована от .py файлов)
MODEL_TYPE   = 'cnn1d'
BATCH_SIZE   = 32
NUM_EPOCHS   = 100
```

## Нормализация

Токи (`~0.07–260 А`) и напряжения (`~100 кВ`) имеют разные масштабы.  
Применяется **per-channel `StandardScaler`** — каждый из 6 каналов нормализуется независимо.  
Расстояние нормализуется `MinMaxScaler` → `[0, 1]`.
